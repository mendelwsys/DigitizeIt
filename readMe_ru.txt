Проект векторизатора растровых изображений.

Программа предназначена для автоматической векторизации растровых изображений и последующему переводу их в shp формат.

Разработано на основе ядра проекта собственной гис-системы (ToyGis).
При запуске производится построение приложения из функциональных программных блоков на основе конфигурации xml.
Имеет два языковых интерфейса, английский (по умолчанию) и русский
Для запуска нужна java!32! версии 7 или 8 только для Windows.

Документация представлена в папке DOC(S) (На русском языке)
Примеры растров для оцифровки находятся в папке MAPDIR.

Процесс векторизации состоит из следующих шагов

1. Выбираем и загружаем растровое изображение.
2. Выделяем цвет на растре, объекты которого будем оцифровывать.
3. Настраиваем параметры оцифровки, в т.ч. параметры фильтрации для оцифровки ( применяется медианный фильтр для растра).
4. Создаем векторный слой, он создается автоматически над растровым слоем, после нажатия кнопки векторизовать.
5. Сохраняем векторный слой в список слоев для перевода в shp файл (в левой части экрана), позволяет так же выбирать стили прорисовки слоя.
6. Выбираем параметры перевода в shp файл.
7. Переводим в *shp - результат создается в папке MAPDIR, во время перевода можно выполнить гео-привязку 
растра с помощью файла описания проекции и гео-координат растра *TAB.
